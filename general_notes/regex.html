Regular Expressions

Some basics about regular expressions (courtesy of http://regexone.com/lesson/)...

  • \d (any digit)
  • the dot . is a wildcard (any single character)
  • escaped dot \. (an actual period)
  • [abc] (match only a single letter a, b or c...nothing else)
  • [^abc] (excludes a single letter a, b, or c)
expression meaning
\d any digit
. wildcard; any single character
\. an actual period
[abc] match only a single letter a, b or c...nothing else
[^abc] excludes a single letter a, b, or c
