Regular Expressions

(courtesy of http://regexone.com/lesson/)...

BASICS
expression meaning
\d any digit
. wildcard; any single character
\. an actual period
[abc] match only a single letter a, b or c...nothing else
[^abc] excludes a single letter a, b, or c
[2-7] any digit from 2 to 7
[^f-n] any letter except those from f to n
ab?c character b is optional (both "ac" and "abc" will match)
\s any whitespace
REPETITIONS
w{3} match the character w exactly 3 times
r{1, 3} match the character r at least once but no more than 3 times
\d* zero or more digits
\d+ 1 or more digits
g+ one or more of the character g
[abc]+ one or more of any a, b, or c character
