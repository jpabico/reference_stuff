Documentation at api.jquery.com

Selectors can be saved 

                    examples:        
                            var paragraph = $("p");
                            var paragraf = $("<p>...</p>")

General set up to use jQuery in a document: 

                    $(document).ready (function() {
                        ---code block ---
                    });
