Lesson Summaries from Shaping Up With Angular (Code School)

Lesson 1.1 -Ramp Up

  1. Angular is a client-side JS framework for adding interactivity to HTML
  2. "How do we tell our HTML when to trigger our JS?"
  3. Works with jQuery & easy to test
  4. A directive is a marker on an HTML tag that tells Angluar to run/reference some JS code (it's how we bind the behavior)
  5. Download the libraries (http://angularjs.org - use angular.min.js)
  6. Bootstrap is also used in the lessons (http://getbootstrap.com - use bootstrap.min.css)
  7. Modules are where pieces of Angular application and dependencies are written to make code more maintainable, testable, and readable
  8. Expressions allow us to insert dynamic values into (see https://docs.angularjs.org/guide/expression)

Lesson 1.2 - Creating a Store Module

  1. Create a module named 'gemStore'
    • (in app.js) var app = angular.module('gemStore', [])
  2. Attach gemStore module to HTML page with a directive and create simple expression to display a message
    • (in index.html) in html tag, ng-app='gemStore'
    • (in index.html) in p tag, {{"Hello, Angular!"}}
